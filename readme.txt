GP
- Himpunan terminal mencakup beberapa variabel (x1 ke xn)
- fungsi set meliputi perkalian, pembagian, pengurangan dan penjumlahan.
- seleksi dilakukan dengan menggunakan pilihan turnamen
- pemilihan titik crossover dilakukan seragam, sehingga setiap simpul yang dipilih memiliki kemungkinan yang sama
- mutasi dilakukan secara acak. jika node terminal diganti dengan terminal yang di acak, jika node fungsi diganti dengan fungsi yang di acak.
